# Roulette-bot-and-builder
The builder let you create strategies for the bot. The bot wil place the bets for you.


This is ONLY for first person roulette.
maybe in the feature i'll add more tables.

- Leave all files in this folder!

- Check if Tesseract is installed to this (Default)folder:
C:\Program Files\Tesseract-OCR\tesseract.exe'

- Create strategies using the Tool (html)
- Set a default strategy (this one will start when the bot is activated!)
- Check to make sure every setting is correct after saving!!
- Export the saved strategies (they will be in your downloads folder, KEEP THEM THERE!!!).

- When creating a new list of strategies, (re)move the old strategy.json file from the download folder.

- Open the First Person Roulette table.

- if you want to track the systems in the tool, hard refresh the html page and click "Enable OCR" before starting the bot.

- Double click the StartBot.bat file to open the bot.

- Now you have 5 seconds to set the Roulette table to full screen (1920x1080).
